# Xinghuojian (星火鉴)

> AI Tool Quality Assessment Framework · 9-Dimensional Scoring

## ✨ Overview

Xinghuojian is a standardized AI tool quality assessment system. Through nine quantitative dimensions, every AI tool receives an S/A/B/C/D/F six-tier rating.

## 📊 Two Modes

| Mode | Metrics | Time | Use Case |
|------|---------|------|----------|
| ⚡ Quick | 5 core | 30s | Daily browsing |
| 🔬 Full | 9 dimensions | 5min | Pre-library review |

## 🚀 Usage

```bash
python3 run_full_eval.py --tool graphify --mode full
python3 spark/quick_eval.py --tool mempalace
```
