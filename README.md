# 星火鉴 (Xinghuojian)

> AI 工具质量评估体系 · 九维打分 · 让好工具被看见

## ✨ 是什么

星火鉴是一套标准化的 AI 工具质量评估体系。通过九大维度的量化评分，为每一个 AI 工具生成 S/A/B/C/D/F 六级评级。

## 📊 两种模式

| 模式 | 指标数 | 耗时 | 适用场景 |
|------|--------|------|----------|
| ⚡ 快速评估 | 5项核心 | 30秒 | 日常浏览快速筛选 |
| 🔬 完整评估 | 9维深度 | 5分钟 | 入库前全面审查 |

## 📐 九维体系

| 维度 | 权重 | 评估标准 |
|------|------|----------|
| 功能性 | 20% | 功能完整度、核心需求满足 |
| 可用性 | 15% | 安装难度、文档质量 |
| 性能 | 10% | 响应速度、资源占用 |
| 稳定性 | 10% | Bug频率、更新维护 |
| 社区活跃 | 15% | GitHub星数、Issue响应 |
| 代码质量 | 10% | 架构设计、代码规范 |
| 安全性 | 10% | 数据隐私、依赖安全 |
| 扩展性 | 5% | 插件机制、API开放性 |
| 创新性 | 5% | 技术独创性 |

## 🎯 评级标准

| 评级 | 分数 | 含义 |
|------|------|------|
| S | 90-100 | 行业标杆 |
| A | 80-89 | 优秀 |
| B | 70-79 | 良好 |
| C | 60-69 | 一般 |
| D | 40-59 | 较差 |
| F | 0-39 | 不推荐 |
